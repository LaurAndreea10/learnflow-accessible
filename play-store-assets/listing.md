## Full description

Linguax Accessible is an inclusive language learning app designed for learners who need a clearer, calmer and more accessible experience.

Learn beginner English, Romanian and basic sign language through short lessons, quizzes, progress tracking and a completion certificate.

Accessibility-first features:

- High contrast mode
- Large text option
- Focus mode
- Reduced motion
- Simplified navigation
- Keyboard-friendly interface
- Voice support
- Image and icon descriptions
- Screen-reader friendly status updates

Learning features:

- English basics
- Romanian basics
- Sign language intro
- Quizzes after lessons
- Local progress tracking
- Exportable progress report
- Completion certificate

Privacy-friendly:

Linguax Accessible works without account creation, without a backend server and without paid services. Progress is stored locally on your device.
