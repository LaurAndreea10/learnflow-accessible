# v1.0.0-google-play-mvp

Release name: **Linguax Accessible: Learn Languages**

## Summary

Google Play MVP for an accessibility-first language learning app.

## Included

- Public rebrand to Linguax Accessible
- Commercial Play Store hero
- English basics learning path
- Romanian basics learning path
- Basic sign language learning path
- Accessibility profile onboarding
- High contrast mode
- Large text mode
- Dyslexia-friendly option
- Reduced motion support
- Focus mode
- Voice support
- RO / EN interface
- Lesson quizzes
- Mistake Review
- Practice again from saved mistakes
- Local progress tracking
- Export progress report
- Import progress report
- Reset progress
- Certificate preview
- Download / print certificate
- Accessibility audit score
- Privacy Policy
- PWA manifest
- Service worker
- Icon SVG assets
- Screenshots folder
- Google Play Data Safety notes
