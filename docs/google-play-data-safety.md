# Google Play Data Safety Notes

App name: **Linguax Accessible: Learn Languages**

## Data collection

The app does not collect personal data.

## Data sharing

The app does not share personal data with third parties.

## Local storage

Learning progress, saved mistakes, selected language and accessibility preferences are stored locally on the user device with browser localStorage.

## Account creation

No account is required.

## Backend

No backend server is used.

## Advertising

No advertising SDK is used.

## Analytics

No third-party analytics SDK is used.

## Voice features

Voice features may use browser or device speech capabilities. The app does not store voice data.

## User controls

Users can reset progress, export progress and import progress from inside the app.

## Suggested Play Console answers

- Data collected: No
- Data shared: No
- Data encrypted in transit: Not applicable because no app data is sent to a server
- Account creation: No
- Users can request deletion: Not applicable for server data; local data can be reset in app
- Ads: No
